# ✉️ EnvelopePrinter - 信封打印工具

一个简洁、强大的在线信封打印工具，支持中国大陆标准信封和国际信封格式。无需安装任何软件，直接在浏览器中设计并打印专业的信封。

## ✨ 主要特性

### 📋 功能亮点

- **🌏 多地区模板支持**
  - 中国大陆标准格式（红色邮编框）
  - 国际/港澳台通用格式

- **📐 灵活的信封规格**
  - DL 标准信封 (220 x 110mm)
  - C5/C6/C7 号信封
  - ZL 国内标准 (230 x 120mm)
  - #10 美式信封 (4.125 x 9.5in)
  - 自定义尺寸支持

- **🎨 可视化设计**
  - 实时预览效果
  - Logo 上传、拖动、缩放
  - 字体大小自由调整
  - 网格对齐辅助线

- **🖨️ 精确打印**
  - 零边距打印支持
  - 自动适配页面尺寸
  - 打印设置向导

## 🚀 快速开始

### 使用方法

1. **直接打开** `index.html` 文件，或部署到任意 Web 服务器
2. **填写信息**：
   - 收件人姓名、地址、邮编（必填）
   - 发件人姓名、地址、邮编（必填）
   - 联系电话、公司/机构等（可选）
3. **调整样式**：
   - 切换地区模板
   - 选择信封规格
   - 调整字体大小
   - 上传并调整 Logo 位置
4. **预览并打印**：
   - 检查预览效果
   - 点击"立即打印"
   - 按照打印向导设置打印机参数

### 系统要求

- **桌面浏览器**（推荐）：
  - Google Chrome 90+
  - Microsoft Edge 90+
  - Firefox 88+
  - Safari 14+

- **不支持移动设备**：手机和平板设备由于屏幕和打印限制，暂不支持使用

## 📖 使用指南

### 基本流程

#### 1. 内容设置

在左侧"内容"标签页中填写信息：

**收件人信息（必填）**
- 姓名/称呼
- 详细地址
- 邮政编码
- 联系电话（可选）
- 公司/机构（可选）

**发件人信息（必填）**
- 姓名
- 地址
- 邮政编码
- 联系电话（可选）
- 公司/机构（可选）

**其他选项**
- 信件主题/备注
- Logo 图片上传

#### 2. 样式调整

在"样式"标签页中：

- **地区模板风格**：选择中国大陆或国际格式
- **字体大小调整**：10-24pt 可选
- **显示选项**：
  - 邮编红框（仅中国大陆模板）
  - 对齐辅助线

#### 3. 页面设置

在"设置"标签页中：

- **信封规格**：选择预设尺寸或自定义
- **预览缩放**：0.5x - 1.5x 调整预览大小（不影响打印）

#### 4. Logo 功能

- **上传**：支持 JPG、PNG、GIF 等常见图片格式
- **拖动**：鼠标拖动 Logo 调整位置
- **缩放**：激活 Logo 后，拖动四角控制点调整大小
- **删除**：点击 Logo 右上角的 × 按钮

### 打印设置指南

点击"立即打印"后，按照以下步骤设置打印机：

1. **选择打印机** - 确认已连接到正确的打印设备
2. **设置纸张尺寸** - 在"更多设置"中选择对应的信封尺寸
3. **页边距设置为 0** - 所有边距设为 0mm 或"无"
4. **关闭页眉和页脚** - 取消勾选"页眉和页脚"
5. **启用背景图形** - 确保 Logo 和样式正常显示

> **提示**：不同浏览器的打印界面可能略有不同，但设置项基本一致。

## 📝 常见问题

### Q: 为什么打印时信封大小不对？

A: 请确保在打印设置中将纸张尺寸设置为与您选择的信封规格一致，并将页边距设为 0。

### Q: Logo 没有打印出来怎么办？

A: 在打印设置中勾选"背景图形"选项。

### Q: 可以保存设计好的信封吗？

A: 当前版本暂不支持保存功能。建议填写完成后直接打印，或使用浏览器的"打印到 PDF"功能保存为 PDF 文件。

### Q: 支持批量打印吗？

A: 当前版本为单个信封设计工具，不支持批量打印。未来可能会添加此功能。

### Q: 为什么移动设备无法使用？

A: 由于打印功能和精确定位需要桌面环境支持，移动设备的浏览器和打印能力受限，因此暂不支持移动端使用。
